
  


 # 📊 Habit Tracker

A modern, interactive habit tracking application built with React and TypeScript. Track your daily habits, visualize your progress, and build better routines with an intuitive, responsive interface.

## 🎯 Overview

**Habit Tracker** is a monthly habit tracking MVP (Minimum Viable Product) designed to help users build and maintain positive habits. The application provides a clean, user-friendly interface for logging daily habits, tracking streaks, and monitoring progress over time.

### 🔗 Design Reference
The original design specification is available at [Figma Design](https://www.figma.com/design/WqkZrZyRLst12lSv3X5B9s/Monthly-Habit-Tracker-MVP)

## ✨ Features

- **Daily Habit Tracking** - Log your habits for each day of the month
- **Progress Visualization** - See your habit streaks and completion rates at a glance
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Interactive UI** - Smooth animations and intuitive user interactions
- **Data Persistence** - Your habits and progress are saved automatically
- **Drag & Drop** - Organize and manage habits easily

- **React 18.3.1** - UI framework

- **Radix UI** - Accessible component library
- **React Router 7.13.0** - Client-side routing
- **React Hook Form 7.55.0** - Form state management
- **Recharts 2.15.2** - Data visualization
- **React DnD 16.0.1** - Drag and drop functionality
- **Date-fns 3.6.0** - Date manipulation utilities
- **Emotion** - CSS-in-JS styling
